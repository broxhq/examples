# @brox/pdf-extractor

Extract structured text and tables from PDF files.

## Install

```bash
brox install @brox/pdf-extractor
```

## Usage

After installing, the agent automatically gains the ability to handle PDFs. Just ask:

> "Pull the line items from invoice.pdf"
> "Summarize the contract in legal-doc.pdf"

## Permissions

- `fs:read` — to read the PDF file
- `exec` — to run the extraction script
